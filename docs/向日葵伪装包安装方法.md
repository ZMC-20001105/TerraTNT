# 向日葵 Ubuntu 25 伪装包安装方法

## 问题现象
- 向日葵图标可见，但点击后报错无法启动
- 依赖 libgconf-2-4 等旧版库在 Ubuntu 25 中已被移除

## 解决方案：创建伪装包

### 步骤 1: 安装 equivs 工具
```bash
sudo apt update
sudo apt install -y equivs
```

### 步骤 2: 创建伪装包配置文件
```bash
cat > /tmp/libgconf-2-4-fake << 'EOF'
Section: misc
Priority: optional
Standards-Version: 3.9.2

Package: libgconf-2-4
Version: 3.2.6-99fake
Provides: libgconf-2-4
Description: Fake package to satisfy sunlogin dependency
EOF
```

### 步骤 3: 构建并安装伪装包
```bash
cd /tmp
equivs-build libgconf-2-4-fake
sudo dpkg -i libgconf-2-4_3.2.6-99fake_all.deb
```

### 步骤 4: 重新安装向日葵
```bash
cd /tmp
sudo dpkg -i sunlogin.deb
sudo apt-get install -f -y
```

### 步骤 5: 启动向日葵
```bash
/opt/apps/com.oray.sunlogin.client/files/bin/sunloginclient
```

或者在应用程序菜单中搜索"向日葵"启动。

## 验证安装
```bash
# 检查伪装包是否安装
dpkg -l | grep libgconf-2-4

# 检查向日葵服务状态
systemctl status runsunloginclient.service

# 检查向日葵进程
ps aux | grep sunlogin
```

## 获取识别码
启动向日葵图形界面后，识别码会显示在主窗口中。
